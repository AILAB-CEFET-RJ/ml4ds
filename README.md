# ml4ds

Small collection of demos exploring ensemble learning techniques on the Beer consumption dataset and a companion student notebook.

## Installation
- Create an isolated environment:
  ```bash
  conda create -n ml4ds python=3.11
  conda activate ml4ds
  ```
- Install the required dependencies:
  ```bash
  pip install -r requirements.txt
  ```

## Contents
- `Classification_students.ipynb` – Guided notebook for students covering classification fundamentals.
- `beer_ensemble_*` scripts – Bagging, boosting, stacking, and nested cross-validation examples using scikit-learn.
- `Beerconsumption.csv` – Dataset used by the demos and notebook.
